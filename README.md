## SHDash

Simple dashboard to start and stop self hosted game servers on portainer

[![SHDash.png](https://i.postimg.cc/xThK9PNk/SHDash.png)](https://postimg.cc/KR7KNBNF)

## Technologies

-   Node.js
-   Next.js
-   Typescript
-   Mongoose

## Installation

```
npm install
```

## Build

```
npm run build
```

## Usage

```
npm start
```
