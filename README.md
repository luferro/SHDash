## SHDash

Simple dashboard to start and stop self hosted game servers on portainer


<img src="https://i.postimg.cc/xThK9PNk/SHDash.png" alt="Dashboard example" />

## Technologies

-   Node.js
-   Next.js
-   Typescript
-   Mongoose

## Installation

```
npm install
```

## Build

```
npm run build
```

## Usage

```
npm start
```
